Have you ever seen a message similar to this?

Error message: "A script on this page is causing Internet Explorer to run slowly"

or...

Error message: "A script on this page is causing Internet Explorer to run slowly. If it continues to run, your computer may become unresponsive. Do you want to abort the script?"

If you have, and you want to turn that off, you can.

This app allows you to change the value of the Windows registry setting that triggers browser prompts that inform the user of a script that is taking a long time to execute. It also allows or encourages the user to stop the execution of the script. This app sets the value to a value that will allow the scripts to run without the message. You can also change it back to the default value.

Eliminate a script is slowing down your browser message prompts.

Reset to original value. Message prompts will appear as normal.

Technical Details

This app has two scripts written in JavaScript that alter the Windows registry.

nomessages.js - This script creates the registry value "HKEY_CURRENT_USER\\Software\\Microsoft\\Internet Explorer\\Styles\\MaxScriptStatements" if it does not exist. It sets the value to 0xfffffff hexadecimal or -1 decimal.

resetmessages.js - This script deletes the registry value "HKEY_CURRENT_USER\\Software\\Microsoft\\Internet Explorer\\Styles\\MaxScriptStatements" if it exists. This is the default value in Windows.

Please read the information found at https://support.microsoft.com/en-us/kb/175500 for more information.

Script Source Code

cdpbutton1: onclick="javascript:runapp('nomessages.js')"

cdpbutton2: onclick="javascript:runapp('resetmessages.js')"
